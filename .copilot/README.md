.copilot — AI & agent artifacts

Purpose
- Store lightweight agent documentation, prompt templates, and a manifest for Copilot/automation use.

Files
- `AGENTS.md` — human-readable agent descriptions and run commands.
- `PROMPTS.md` — ready-to-use prompt templates for asking the assistant to run tasks.
- `manifest.json` — simple machine-friendly list of known agents and scripts.

Workflow
- Edit `PROMPTS.md` to reflect how you want the AI to behave.
- Use the manifest when wiring external automation or editors to trigger scripts.

If you want, I can also add a `playbook.md` that maps common troubleshooting flows (e.g., audio silent on hardware) to the agents and scripts.
