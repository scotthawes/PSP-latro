# Project Documentation

This folder is organized by purpose to keep docs easy to scan.

## Structure

- `guides/` — how-to and workflow documentation.
- `trackers/` — status trackers and changelogs.
- `plans/` — test plans and validation matrices.
- `reports/` — issue collations and comparison reports.
- `checklists/` — focused verification checklists.
- `issues/ui-parity/` — pregame/ingame parity issue documents.
- `issues/ui-comparison/` — post-parity UI comparison issue documents.
- `references/psxplace/` — archived external reference notes.

## Key Entry Points

- `guides/devguide-psp3000.md` — PSP-3000 development workflow.
- `guides/copilot-guide.md` — Copilot/AI usage guidance.
- `trackers/consolidated-status.md` — consolidated project status and priorities.
- `plans/ui-parity-test-plan.md` — parity validation plan.
- `plans/ui-comparison-test-plan.md` — post-parity comparison plan.
- `reports/ui-parity-issue-collation.md` — parity issue rollup.
- `reports/ui-comparison-issue-collation.md` — remaining comparison issue rollup.
